    java-tron Release History¶

    This page lists every public java-tron release, ordered from newest to oldest. Each row corresponds to a single release and includes:

    • Code Name — Release codename (the GreatVoyage series is named after philosophers; early Odyssey/Exodus releases have none)
    • Version — Full release tag, e.g. GreatVoyage-v4.8.1
    • Released — Date the Git tag was published
    • Incl TIPs — TRON Improvement Proposals that landed in this release
    • Release Note — Link to the GitHub Release Note with the full change list and binary downloads
    • Specs — A dedicated page describing protocol or mechanism changes (only available for selected GreatVoyage v4.0+ releases); for other releases, refer to the Release Note

    Only rows whose Specs column shows a link have a dedicated page. Releases without a dedicated page are typically patch releases (no new mechanism) or earlier versions.
